***client & server

*Node. js is a server-side JavaScript run-time environment.
It's open-source, including Google's V8 engine, libuv for cross-platform compatibility, and a core library.

client->
The Node. js Client API enables you to create Node. js applications that can read, write, and 
query documents and semantic data in a MarkLogic database.

server->
Node. js is an open source server environment. Node. js uses JavaScript on the server. 
The task of a web server is to open a file on the server and return the content to the client.
